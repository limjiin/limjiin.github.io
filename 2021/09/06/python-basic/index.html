변수와 저장 (variable and define)

변수의 종류(dtype)

  • 정수 int
  • 실수 float
  • 문자열 str
  • 불리언 : true, false
  • 객체 : class, model
  • 문자열 저장은 ‘’, “” 로 감싼 텍스트로 가능하다.
  • input(‘입력받을 데이터의 설명:’)

    입력받은 결과를 변수에 함께 저장
    입력받은 결과 또한 문자형식으로 저장
    숫자형태로 인식하기 위해 데이터 타입 변환이 필요함

변수 값의 형변환(type cating)

int(), float(), str() 명령어를 사용

  • 문자열 > 실수 > 정수: 반대의 경우는 조건이 생김

출력 (print)

소괄호 안에 숫자, 따옴표로 감싼 문자, 변수 가
따옴표는 작은, 큰 모두 가능
New format string
파이썬 3.6버전 이후 권장하는 출력방식
기존 format string 보다 사람이 보기에 직관적이고 명확한 문자와 변수간 관계 확인이 가능하다.

  • print(f’{양name}의 키는 {hight}cm이고 몸무게는 {weight}kg입니다. 직업은 {job}입니다.’)

연산

  • + : 더하기
  • - : 빼기
  • * : 곱하기
  • ** : 거듭제곱
  • / : 나누기
  • // : 몫
  • % : 나머지

조건 연산

  • and, & : 동시충족
  • or, | : 조건충족
  • in : 보유여부 (멤버연산자라 부릅니다)
  • is : 같음여부
  • not : 다름여부
  • == : 같은값인지 확인
  • !=: 다른값인지 확인
  • <, >, <=, >= : 대소비교
  • bool 타입으로 결과값을 내준다.
  • 참인지 거짓인지 True합, False로 구분가능한 데이터

조건 곱, 조건 합

0 : False, 1 : True
and : 곱연산, or : 합연산
