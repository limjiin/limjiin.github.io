python-sequence

Posted on 2021-09-08 In python Disqus:

파이썬의 기초 개념을 정리해보았다.
2편은 sequence(list, tuple, dict 등)를 정복한다!
(개인적으로 재미 없,,,,)

자료 구조

list, tuple, dict, set

list

순서가 있다.
[ ] 대괄호로 묶어 사용한다.
리스트 안에 들어가는 요소들은 ,(쉼표)로 구분한다.
리스트 내부에는 정수, 실수, 문자열 뿐만 아니라 자료구조인 list 와 기타 자료구조도 포함 가능

리스트 갯수 세기

  • len()

리스트의 인덱싱(indexing), 슬라이싱(slicing)

리스트의 특정 항목에 접근(색인)
인덱싱은 하나의 값에 접근, 슬라이싱은 여러개의 값의 묶음에 접근
리스트의 순서를 인덱스라고 하며 0부터 시작

  • 리스트 전체 항목 중 2의 배수 인덱스 항목 가져오기: wallet[::2]

리스트 편집

내부 명령어
리스트는 자료구조를 갖고 있으면서 리스트 작업을 위해 필요한 내부명령어도 함께 제공
리스트이름 뒤 .(콤마)로 명령어에 접근 가능

  • list_name.func()
  • .append() : 추가할 값 전달하여 리스트에 추가
  • .remove() : 특정 항목 삭
  • .pop() : 마지막 항목 삭제
  • .reverse() : 정렬 없이 역순
  • .count() : 숫자

리스트 연산어

  • max(값), min, sum, 값 in제 변수
  • 뎃셈 연산: num1 + num2
  • 곱셉 연산: num1 * 2
  • 조건연산자 테스트: 1 in num1

리스트 삭제

  • .clear() : 리스트는 남아있지만 내부 항목은 모두 삭제합니다.
  • del

tuple

연산이나 입력값을 전달하는 자료구조로는 잘 사용하지 않고 결과값을 출력하는 경우 많이 사용한다.
( ) 소괄호로 묶어 사용한다.
구분은 , 콤마로 사용한다.

인덱스 불가 : 한 번 튜플에 들어간 값은 변경이 불가함
값 추가는 가능하다.

  • del : 전체 삭제만 가능하다.

dict

key : value
{ } 중괄호로 묶어 사용한다.
딕셔너리 요소 구분은 리스트와 마찬가지로 쉼표
중괄호 내 key : value 값을 전달하고 각 항목의 구분은 쉼표

딕셔너리 갯수 세기

  • len()

딕셔너리 편집

  • 딕셔너리에 point card key를 갖는 해피포인트 문자열을 값으로 저장

wallet['point card'] = '해피포인트'
wallet
  • 딕셔너리 특정 항목 업데이트

wallet.update({'point card':'CJ one'})
wallet
  • 딕셔너리 내부 속성 값에 접근해서 값 업데이트도 가능하다.
  • wallet의 coin key값을 갖는 값에 50원:1 값을 추가한다.

wallet['coin']['50원'] = 1
wallet
  • 딕셔너리 특정 value값 빼오기

wallet.pop(100)
wallet

딕셔너리 삭제

  • del : 항목 제거
  • .clear() : 전체 삭제

딕셔너리 추가 명령

리스트와 마찬가지로 딕셔너리 내부에 리스트등 타 자료구조 저장 가능

  • .keys() : 키 값을 확인
  • .values() : 값을 확인
  • .items() : key, value 쌍 확인

set

데이터셋의 고유값을 확인하는 용도로도 사용(중복제거)
{ } 대괄호로 묶어 사용한다.

셋 편집

셋 값 하나삭제


test_set1.discard(5)
test_set1

셋에 값 하나 추가

  • 중복을 허용하지 않기 때문에 셋에 같은 값이 있을 경우 추가하지 않음

test_set1.add(5)
test_set1
  • 셋에 값 여러개 추가

test_set1.update({5, 6, 7})
test_set1

집합 연산을 지원한다.

  • .update() : 여러 개 추가
  • .discard() : 하나 삭제
  • .intersect() : 교집합
  • .union() : 합집합
  • .difference() : 차집합
  • and, or 값도 활용 가능

셋 삭제

  • del
